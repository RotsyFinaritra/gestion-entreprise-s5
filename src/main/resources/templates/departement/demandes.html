Comment ça marche ?
  1. Créez une demande en précisant le poste et les détails
  2. Votre demande est envoyée à l'équipe RH avec le statut "En attente"
  3. Les RH examinent votre demande et peuvent l'accepter ou la refuser
  4. Si acceptée, l'offre est publiée sur les canaux de recrutement
Conseils pour une demande efficace
  • Justifiez clairement le besoin de recrutement
  • Fournissez une description détaillée du poste
  • Indiquez une fourchette salariale réaliste
  • Mentionnez la priorité si le besoin est urgent
